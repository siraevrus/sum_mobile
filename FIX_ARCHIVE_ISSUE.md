# Решение проблемы: "Archive" не активна в Xcode

## Проблема
В меню Product → Archive недоступен (неактивен).

## Решение (выберите подходящий вариант):

### Вариант 1: Выберите правильное устройство в Xcode
1. В верхней панели Xcode найдите выпадающий список рядом с именем схемы "Runner"
2. **ВЫБЕРИТЕ "Any iOS Device"** (НЕ симулятор!)
3. После выбора Archive станет активным

### Вариант 2: Подключите физическое устройство
1. Подключите iPhone/iPad через USB
2. Доверьте компьютеру на устройстве
3. Выберите устройство в списке устройств
4. Archive станет активным

### Вариант 3: Проверьте настройки схемы
1. В Xcode: Product → Scheme → Edit Scheme
2. Выберите "Archive" слева
3. Убедитесь что "Build Configuration" = "Release"
4. Закройте окно

### Вариант 4: Соберите через командную строку
Выполните в терминале:
```bash
cd /Users/ruslansiraev/sum_mobile
flutter clean
flutter pub get
flutter build ipa --release
```

Результат будет в: `build/ios/ipa/`

## Частая причина
"Archive" неактивна когда выбран симулятор вместо устройства!
