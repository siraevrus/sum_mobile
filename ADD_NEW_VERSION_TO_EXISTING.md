# Добавить новую версию в существующее приложение

## Проблема
Приложение "Wood Warehouse" уже существует в App Store Connect.

## Решение: Обновите существующее приложение

### Шаг 1: Найдите приложение в App Store Connect
1. Откройте https://appstoreconnect.apple.com
2. Перейдите в "Мои приложения"
3. Найдите "Wood Warehouse"
4. Откройте приложение

### Шаг 2: Проверьте Bundle ID
В настройках приложения убедитесь что Bundle ID:
**ru.expwood.Wood.Warehouse.**

### Шаг 3: Добавьте новую версию
1. На главной странице приложения
2. Нажмите **"+ Version"** или "+ Что нового в этой версии"
3. Выберите платформу iOS
4. Введите номер версии: **1.0.3**
5. Выберите загруженный билд (после обработки)
6. Заполните описание изменений

### Шаг 4: Загрузите билд через Xcode
1. Откройте Xcode → Window → Organizer
2. Выберите архив "Runner" (от сегодня)
3. Нажмите "Distribute App"
4. Выберите "App Store Connect"
5. Нажмите "Upload"
6. Дождитесь завершения

### Шаг 5: Дождитесь обработки
1. После загрузки в App Store Connect появится уведомление
2. Обработка займет 10-30 минут
3. Когда билд будет готов, он появится в списке доступных

### Шаг 6: Отправьте на ревью
1. Выберите загруженный билд в версии
2. Заполните все обязательные поля
3. Добавьте скриншоты если нужно
4. Нажмите "Отправить на ревью"

## Важно!
- НЕ создавайте новое приложение!
- Используйте существующее "Wood Warehouse"
- Просто добавляйте новую версию
