# Как переименовать проект iOS с "Runner" на "Wood Warehouse"

## Важно! 
**"Runner" - это техническое имя проекта в Xcode. Менять его нужно аккуратно!**

## Что УЖЕ настроено правильно:
✅ CFBundleDisplayName = "Wood Warehouse" (показывается под иконкой на устройствах)
✅ CFBundleName = "Wood Warehouse" (только что обновлено)

## Что МОЖНО переименовать:

### Вариант 1: Переименовать в Xcode (РЕКОМЕНДУЕТСЯ)
1. Откройте `ios/Runner.xcworkspace` в Xcode
2. Выберите проект "Runner" в навигаторе слева
3. Нажмите Enter или F2 на названии
4. Переименуйте в "Wood Warehouse"
5. Подтвердите все изменения
6. Xcode автоматически обновит все ссылки

### Вариант 2: Ручное переименование (НЕ рекомендуется)
Это может сломать проект. Лучше использовать Xcode.

## Итог:
- На устройствах приложение УЖЕ называется "Wood Warehouse"
- "Runner" - это только внутреннее имя проекта
- Можно оставить как есть, или переименовать через Xcode

## Текущая настройка:
- Отображаемое имя (на устройствах): ✅ "Wood Warehouse"
- Имя проекта в Xcode: "Runner"
